# Unity3D_Loft
**由于Github无法上传大于100M的资源文件，只好写一下项目总结了**

WPF框架下的OSG三维地理信息系统显示地球，而Unity3D制作小场景用于显示特定小场景，通过接入监控视频流，传感器接口用于实现小场景的展示及监控。
整体软件界面，右上角为Unity3D小场景。
